In this introductory level project, you'll use Sinatra to build a number guessing game.

Goals\n
It will:

Generate a random number on startup\n
Have a web page where you can enter a guess\n
Tell you whether your guess was...
-  Way too high
-  Too high
-  Correct
-  Too low
-  Way too low
